# 成長株分析システム「Shintakane」

## はじめに

本システムは、日本株式市場における**成長株の発掘・分析・ランキング**を自動化するシステムです。毎日の市場データを収集し、独自の多角的スコアリングで有望銘柄を順位付けします。

投資判断において「どの銘柄に注目すべきか」を効率的に絞り込むことが本システムの目的です。

---

## システムが行うこと

### 1. 毎日の銘柄スクリーニング

平日19時に自動で以下の処理が実行されます。

- **新高値銘柄の検出**: 52週間の最高値を更新した銘柄を自動収集
- **出来高急増銘柄の検出**: 通常より大幅に出来高が増加した銘柄を検出
- **データベースの更新**: 約300銘柄のウォッチリストに対して、最新の株価・業績・指標を自動取得
- **総合ランキングの生成**: すべての銘柄を独自スコアで順位付けし、CSV形式で出力

### 2. データの自動収集源

| データ元 | 取得する情報 |
|----------|------------|
| 株探（Kabutan） | 新高値・出来高急増銘柄、四半期/年次業績、バリュエーション指標、企業テーマ |
| Yahoo Finance Japan | 日次の株価データ（始値・高値・安値・終値・出来高） |

### 3. 結果の配信

分析結果はCSVファイルとして出力され、Google Driveへ自動アップロードされます。スプレッドシート等ですぐに確認できます。

## 独自のスコアリングシステム

本システムの核心は、4つの評価軸を組み合わせた**総合スコア（0〜100点）**です。

### 総合スコアの構成

```
総合スコア ＝ 業績（40%）＋ 指標（20%）＋ モメンタム（25%）＋ ファンダメンタル（15%）
```

| 評価軸 | 配分 | 見ているもの |
|--------|------|------------|
| **業績（Profit/Quality）** | **40%** | 過去5年間の増収増益の一貫性、直近四半期の成長率 |
| **モメンタム（Momentum）** | **25%** | 株価の相対的な強さ（TOPIX対比） |
| **指標（Value/Size）** | **20%** | 時価総額の適正さ、財務健全性（有利子負債比率） |
| **ファンダメンタル（Fundamental）** | **15%** | 市場で注目されている投資テーマとの関連性 |

---

### 各評価軸の詳細

#### 業績スコア（40%）— 成長の質と持続性

最も重視している評価軸です。一時的な好業績ではなく、**継続的に成長している企業**を高く評価します。

- **過去5年間の成長実績**（20点満点）
  - 年率7%以上の利益成長で満点、4%以上で半分の評価
  - 売上高の年率成長率（CAGR）も加味
  - 利益がマイナスだった年があれば減点

- **直近の利益成長率**（10点満点）
  - 直近年度の営業利益が前年比20%以上で満点

- **四半期ごとの成長トレンド**（最大70点）
  - 直近3四半期の売上・利益の成長率を個別に評価
  - 四半期単位で20%以上の利益成長、10%以上の売上成長があるほど高得点

**ポイント**: 「たまたま良い年」ではなく、四半期・年度を通じて安定成長している企業が高スコアになります。

#### モメンタムスコア（25%）— 市場からの評価

株価が市場平均（TOPIX）と比べてどれだけ強いかを統計的に評価します。

- **相対力指数（RS）の算出**: 13週・26週・39週・52週の株価変動率を加重平均
- **TOPIX対比の正規化**: TOPIXのRS値と比較し、正規分布を用いて上位何%にいるかをスコア化

**ポイント**: 単純な株価上昇率ではなく、「市場全体の動きと比べてどれだけ強いか」を測定します。市場全体が上がっている中での上昇は低評価、市場が下がっている中での底堅さは高評価になります。

#### 指標スコア（20%）— 投資適格性

時価総額と財務の健全性から、投資対象としての適格性を判定します。

- **時価総額の評価**（30点満点）
  - 最適ゾーン: 時価総額400億〜1,000億円（成長余地と流動性のバランス）
  - 小さすぎる（100億円未満）＝ 流動性リスク → 低得点
  - 大きすぎる（3,000億円超）＝ 成長鈍化の可能性 → 低得点

- **財務健全性**（±30点）
  - 有利子負債自己資本比率で判定
  - 0.5倍以下 ＝ 健全 → 加点
  - 3.0倍超 ＝ 財務リスク → 大幅減点

- **理論株価との乖離**
  - 5年間の予想利益から算出した理論株価との比較

**ポイント**: 日本市場の特性として、中型株（時価総額400億〜1,000億円）に最も大きな成長機会があるという考えに基づいた設計です。

#### ファンダメンタルスコア（15%）— 旬のテーマ

各銘柄が属する投資テーマ（AI、半導体、ヘルスケア等）の市場での注目度をスコア化します。

- 株探のアクセスランキングから日々のテーマ人気度を取得
- 人気テーマに属する銘柄ほど高スコア

**ポイント**: ファンダメンタルズだけでなく「今、市場が何に関心を持っているか」というセンチメント要因も取り入れています。

## テクニカル分析指標

スコアリング以外にも、以下の補助指標をレポートに含めています。

### トレンドテンプレート（7項目チェック）

機関投資家の買い集め局面に典型的な7つの条件をチェックします。

1. 株価が30週移動平均線＆40週移動平均線の上にある
2. 30週移動平均線が40週移動平均線の上にある
3. 40週移動平均線が1ヶ月以上上昇トレンド
4. 10週移動平均線が30週・40週移動平均線の上にある
5. 株価が10週移動平均線の上にある
6. 52週安値から30%以上上昇＆52週高値から25%以内にある
7. 相対力指数（RS）が70以上

**7点中何点か**で、上昇トレンドの質を一目で判断できます。

### 売り圧力レシオ（SPR）

日々の値動きと出来高から、買い勢力と売り勢力のバランスを推定します。

- 20日SPR / 5日SPR: 長期・短期の需給バランス
- 値が低いほど買い優勢（40%以下が目安）

### ポケットピボット

移動平均線付近で、過去の下げ日の出来高を上回る高出来高の上昇が発生した場合のシグナル。機関投資家の静かな買い集めを示唆するパターンです。

### ローソク足ボラティリティ

日中の値動きの大きさ（20日平均 / 5日平均）。ボラティリティが高まっている銘柄の識別に使います。

## レポートの見方

### 総合ランキング（code_rank.csv）

毎日更新されるメインのレポートです。主な項目：

| 列名 | 内容 |
|------|------|
| 順位 | 総合スコアに基づく順位 |
| コード / 銘柄名 | 証券コードと企業名 |
| セクター | 業種分類 |
| 総合PT | 4軸の加重平均による総合スコア |
| プロフィット/クォリティ | 業績スコア（40%配分） |
| バリュー/サイズ | 指標スコア（20%配分） |
| モメンタム | モメンタムスコアと推移（25%配分） |
| ファンダメンタル | テーマスコア（15%配分） |
| シグナル | ポケットピボット等のテクニカルシグナル |
| トレンドテンプレート | 7項目チェック結果 |
| 売り圧力レシオ | 需給バランス指標 |
| 業績 | 今期/四半期の売上・営利成長率 |
| 指標 | 時価総額・PER・PSR・ROE等 |
| 理論株価 | 乖離率と上限/下限 |
| テーマ | 関連する投資テーマ |

### 日次発掘レポート（shintakane_result.csv）

その日に新高値や出来高急増を記録した銘柄の分析結果。「今日注目すべき銘柄」のリストです。

## システムの設計思想

### なぜ業績を最重視するのか（40%）

短期的な株価変動ではなく、**企業の実力（＝利益の持続的成長）**を最も重要な判断基準としています。過去5年間の一貫した増収増益を評価するため、一過性の特需や会計上の操作では高スコアを得られません。

### なぜモメンタムを含めるのか（25%）

業績が良くても株価が反応していない銘柄は、まだ市場に認知されていないか、何かの理由で評価されていない可能性があります。モメンタムを加味することで、**市場が実際に資金を投じ始めている銘柄**を優先的に捉えます。

### なぜ時価総額を評価するのか（20%）

時価総額400億〜1,000億円の中型株ゾーンは、成長余地が大きく、かつ個人投資家が参入可能な流動性を持つ領域です。小型すぎる銘柄の流動性リスクと、大型すぎる銘柄の成長鈍化リスクの両方を回避します。

### 毎日の自動化

平日19時に自動実行される設計は、東京証券取引所の引け後（15:30）に十分な時間を置いてからデータ取得を行うためです。15:30の引け後に各種データが更新されるのを待ち、安定した最新データを取得します。

## 開発者について

本システムは、個人投資家としての実践的なニーズから生まれた**完全自作のシステム**です。

### 技術的な特徴

- **Python製の自動分析パイプライン**: データ取得からスコアリング、レポート出力、クラウド配信まで全工程を自動化
- **複数データソースの統合**: 株探・Yahoo Finance Japanから、業績・株価・テーマなど多面的なデータを収集
- **統計的手法の活用**: 相対力指数の正規分布正規化など、定量的なアプローチでバイアスを排除
- **堅牢なシステム設計**:
  - 4段階のキャッシュ管理でWebスクレイピングの負荷を最小化
  - 決算発表日の自動検知と強制データ更新
  - マルチスレッド対応のデータベース設計
  - 自動テスト（CI/CD）によるシステムの品質保証
- **継続的な改善**: GitHubでのバージョン管理、プルリクエストベースの開発、自動テストの整備

### 投資哲学とシステムの関係

このシステムは、以下の投資哲学を定量的に実装したものです。

1. **ファンダメンタルズ重視**: 利益成長の一貫性を最優先する（全体の40%）
2. **マーケットタイミングの考慮**: 業績だけでなく、株価が実際に動き始めているかを確認する（25%）
3. **リスク管理**: 財務健全性と適正な時価総額を重視し、投機的な銘柄を排除する（20%）
4. **市場センチメントの活用**: 旬のテーマを捉えつつ、比重は控えめにして振り回されないようにする（15%）

人間の感覚や直感に頼らず、明確な基準で銘柄を評価することで、**感情的なバイアスを排除した合理的な投資判断**を支援するシステムです。

## 注意事項

- 本システムは投資判断の**参考情報**を提供するものであり、特定銘柄の売買を推奨するものではありません
- 過去の業績データに基づくスコアリングであり、将来の株価変動を保証するものではありません
- 投資に関する最終判断は、ご自身の責任において行ってください
