# テーマ急上昇ニュース調査

market_data.csv のテーマランク上位3テーマについて、急上昇の理由となる市場ニュース・材料をWebSearchで調査する。

## 手順

### 1. テーマランクデータの読み取り

`data/code_rank_data/market_data.csv` を読み込み、以下を抽出する:

- 「■ テーマランク」セクションを探す
- 「ランク」で始まる行から、2列目以降の上位3テーマ名を取得
- 「騰落率」で始まる行があれば、対応する3テーマの騰落率データ（`+X.X%[N]` 形式）も取得

### 2. 各テーマのニュース検索

上位3テーマそれぞれについて、WebSearchで以下のクエリを実行:

```
"{テーマ名}" 株式市場 材料 ニュース {現在の年月}
```

検索は日本語で行い、速報性のある市場ニュース・材料に絞る。各テーマ1回の検索で十分。

### 3. 結果の出力

以下のフォーマットで出力する:

```
## テーマ急上昇ニュース（{今日の日付}）

### 1. {テーマ名}（騰落率: {騰落率データ or "データなし"}）
- {ニュース要約1}
- {ニュース要約2}

Sources:
- [記事タイトル1](URL1)
- [記事タイトル2](URL2)

### 2. {テーマ名}（騰落率: {騰落率データ or "データなし"}）
...

### 3. {テーマ名}（騰落率: {騰落率データ or "データなし"}）
...
```

## 注意事項

- 各テーマにつき2-3件のニュース・材料を簡潔に要約する
- 出典URLを必ず付記する（Sourcesセクション）
- CSVファイルが存在しない、またはテーマランクセクションがない場合はその旨を伝える
- 検索結果が見つからない場合は「関連ニュースが見つかりませんでした」と記載する
